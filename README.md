# Learning

### How to compile/interpret and run the code?
<ul>
  <li>
    How to run Bash file?
    </br>
    ->
    <code>
      bash ./file_name.bash
    </code>
  </li>
  <li>
    How to compile and run C file?
    </br>
    <ol>
      <li>
        <code>
          gcc -o file_name ./file_name.c
        </code>
      </li>
      <li>
        <code>
          ./file_name
        </code>
      </li>
    </ol>
  </li>
  <li>
    How to compile and run C# file?
    </br>
    <ol>
      <li>
        <code>
          mcs -out:file_name.exe file_name.cs
        </code>
      </li>
      <li>
        <code>
          ./file_name.exe
        </code>
      </li>
    </ol>
  </li>
  <li>
    How to compile and run C++ file?
    </br>
    <ol>
      <li>
        <code>
          g++ -o file_name ./file_name.c
        </code>
      </li>
      <li>
        <code>
          ./file_name
        </code>
      </li>
    </ol>
  </li>
  <li>
    How to compile and run Haskell file?
    </br>
    <ol>
      <li>
        <code>
          ghc file_name.hs
        </code>
      </li>
      <li>
        <code>
          ./file_name
        </code>
      </li>
    </ol>
  </li>
  <li>
    How to run Javascript file?
    </br>
    ->
    <code>
      node ./file_name.js
    </code>
  </li>
  <li>
    How to run Python 3 file?
    </br>
    ->
    <code>
      python3 ./file_name.py
    </code>
  </li>
  <li>
    How to run Typescript file?
    </br>
    ->
    <code>
      node ./file_name.ts
    </code>
  </li>
</ul>
